# expense-tracker-api
# Expense Tracker API

A production-style backend project built using FastAPI and PostgreSQL while learning backend engineering concepts.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy ORM
- Pydantic

## Features

### V1
- Category CRUD
- Expense CRUD
- Database Relationships
- Filtering APIs
- Summary APIs

### Upcoming

#### V2
- JWT Authentication
- User Accounts
- Alembic Migrations

#### V3
- Docker
- Deployment

#### V4
- Redis
- Caching
- Rate Limiting

#### V5
- Background Jobs
- Monthly Report Generation
